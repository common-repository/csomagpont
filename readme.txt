== Description ==

Add fel egyszerűen, a WooCommerce rendelési felületéről vevőid csomagjait a következő futárszolgálatokkal, kedvezményes árakon:

* MPL
* GLS
* Fáma
* DPD
* 5PL Csomagpont

és még rengeteg más futárszolgálat és szállítási opció.

Választhatod budapesti átvevőhelyeink egyikét is, ahol ügyfeleid személyesen átvehetik rendeléseiket.

Teljeskörű logisztikai – fulfillment szolgáltatást is nyújtunk webáruház tulajdonsoknak. 

Nincs még velünk szerződésed? A következő linkre kattintva pillanatok alatt partnerünk lehetsz, és élvezheted a 12 szerződött futárszolgálatunk által nyújtott előnyöket. 

[Ajánlatkérés](https://csomagpont.com/ajanlatkeres/#ajanlat)




A bővítmény a Mav-IT Deliveo szoftverrel való összekötést biztosítja, így annak szolgáltatására támaszkodik (deliveo.nanosoft.hu API).
Bővebb információ a Deliveoról: https://deliveo.eu/hu/
A Mav-IT Kft. Adatkezelési Tájékoztatója: https://mav-it.hu/adatkezelesi-tajekoztato 

= Features =

* Csomagok elküldése egy kattintással a Rendelések felületről
* Alapértelmezett szállátási opció beállítások
* Csomagcímkék, feladójegyzékek tömeges letöltése a Rendelések felületről
* Csomagátvételi pontok támogatása a Szathmári Pont WooCommerce pluginnal


== Installation ==
1. Telepítsd és kapcsold be a bővítményt
2. Add meg a csatlakozáshoz szükséges adatokat a WooCommerce -> Csomagpont menüpontban
3. Add fel rendeléseidet a WooCommerce -> Rendelések menüben
